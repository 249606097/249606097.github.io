树 Tree

Posted on 2020-02-25 Edited on 2020-02-27

树状是一种数据结构，它是由有限结点组成一个具有层次关系的集合。把它叫做“树”是因为它看起来像一棵倒挂的树，也就是说它是根朝上，而叶朝下的。

简介

树是由结点和边组成的且不存在着任何环的一种数据结构。一棵树包括一个根结点和有限个结点，所有结点构成一个多级分层结构。把它叫做“树”是因为它看起来像一棵倒挂的树，也就是说它是根朝上，而叶朝下的。
